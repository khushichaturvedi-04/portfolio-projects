# Portfolio Projects

Hi! I'm Khushi Chaturvedi, a senior in Computer Engineering at North Carolina State University, concentrating in AI and Machine Learning.

This repository is a collection of engineering projects spanning embedded systems, machine learning, data visualization, mobile development, computer architecture, and quantum computing research. Each project folder contains a detailed README covering system design, technical approach, and outcomes.

## Skills and Technologies

**Languages:** C, C++ (C++17), MSP430 Assembly, Python, R, Swift, JavaScript, HTML/CSS

**Embedded and Hardware:** Bare-metal firmware development, real-time/multi-threaded program design, I2C sensor integration, PWM motor control, Wi-Fi/IoT connectivity, REST API design, IAR Embedded Workbench, Arduino/ESP32, hardware debugging (oscilloscope, logic analyzer), ARKit, SceneKit

**Machine Learning and Data Science:** scikit-learn, TensorFlow/Keras, Qiskit, Hugging Face Transformers (BERT), Tableau, ggplot2, dplyr, K-Means clustering, linear and Ridge regression, logistic regression, correlation and regression diagnostics, exploratory data analysis, data wrangling

**Systems and Parallel Computing:** OpenMP, the Structural Simulation Toolkit (SST), MESI/MOESI cache coherence protocols, HPC batch job automation, genetic algorithms (DEAP)

**Tools and Platforms:** Git and GitHub, Google Colab, RStudio and Posit Cloud, Xcode, OpenRocket

## Projects

a. **[Autonomous Embedded Vehicle](https://github.com/khushichaturvedi-04/portfolio-projects/blob/main/ece306-embedded-vehicle/README.md)** — Designed and built an autonomous MSP430-based vehicle in C, implementing line-following navigation, PWM motor control, and IoT telemetry; debugged with oscilloscope and logic analyzer in IAR Embedded Workbench.

b. **[Sprout Sense: Smart Plant Monitoring System](https://github.com/khushichaturvedi-04/portfolio-projects/blob/main/sprout-sense/README.md)** — Senior design capstone. Full-stack IoT plant health monitor built on ESP32, integrating I2C sensors, Wi-Fi connectivity, and cloud logging with a real-time web dashboard.

c. **[Parallel Programming and Computer Architecture Simulation](https://github.com/khushichaturvedi-04/portfolio-projects/blob/main/parallel-computing-architecture/README.md)** — OpenMP parallel programming and an SST-based cache coherence protocol extension from MESI to MOESI.

d. **[Quantum Error Mitigation](https://github.com/khushichaturvedi-04/portfolio-projects/blob/main/quantum-error-mitigation/README.md)** — Undergraduate research on post-processing techniques to correct errors in NISQ (Noisy Intermediate-Scale Quantum) computing.

e. **[Hyperspectral Filter Array Optimization via Genetic Algorithms](https://github.com/khushichaturvedi-04/portfolio-projects/blob/main/hyperspectral-filter-optimization/README.md)** — Ongoing undergraduate research, Dr. Dror Baron's Lab, NC State. Genetic-algorithm-based spectral filter optimization for compressive hyperspectral imaging.

f. **[City Transportation Planner](https://github.com/khushichaturvedi-04/portfolio-projects/blob/main/ece309-transportation-planner/README.md)** — Graph-based shortest-path planner handling 10,000+ stops, built in C++17 using Dijkstra's algorithm.

g. **[AR4D Measure](https://github.com/khushichaturvedi-04/portfolio-projects/blob/main/ar4d-measure/README.md)** — Open-source iOS AR app reimagining spatial measurement as a 4D capture experience, built in Swift, ARKit, SceneKit, and SwiftUI.

h. **[Applied Machine Learning: Global Internet Adoption](https://github.com/khushichaturvedi-04/portfolio-projects/blob/main/dsa412-machine-learning/README.md)** — Capstone applying K-Means clustering and Ridge regression, culminating a semester of supervised, unsupervised, and deep learning coursework.

i. **[Exploratory Data Analysis and Statistical Modeling in R](https://github.com/khushichaturvedi-04/portfolio-projects/blob/main/dsa406-statistical-analysis/README.md)** — Exploratory data analysis, correlation analysis, and linear regression modeling in R.

j. **[Olympic Success Around the World](https://github.com/khushichaturvedi-04/portfolio-projects/blob/main/olympics-data-visualization/README.md)** — Data visualization project examining Olympic medal trends, built in Tableau and R.

k. **[Level 1 High-Power Rocketry Certification](https://github.com/khushichaturvedi-04/portfolio-projects/blob/main/level1-rocketry-certification/README.md)** — NAR/Tripoli Level 1 certification flight, designed and flight-simulated in OpenRocket.
