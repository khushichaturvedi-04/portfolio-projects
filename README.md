# Portfolio Projects

Hi, I am Khushi, a senior in computer engineering, North Carolina State University.

A collection of engineering projects spanning embedded systems, machine learning, data visualization, mobile development, computer architecture, and quantum computing research. Each project folder contains a detailed README covering the system design, technical approach, and outcomes.

## Skills and Technologies

**Languages:** C, C++ (C++17), MSP430 Assembly, Python, R, Swift, JavaScript, HTML/CSS

**Embedded and Hardware:** Bare-metal firmware development, real-time/multi-threaded program design, ARKit, SceneKit, I2C sensor integration, PWM motor control, Wi-Fi/IoT connectivity, REST API design, IAR Embedded Workbench, Arduino/ESP32, hardware debugging (oscilloscope, logic analyzer)

**Machine Learning and Data Science:** scikit-learn, TensorFlow/Keras, Qiskit, Hugging Face Transformers (BERT), Tableau, ggplot2, dplyr, K-Means clustering, linear and Ridge regression, logistic regression, correlation and regression diagnostics, exploratory data analysis, data wrangling

**Systems and Parallel Computing:** OpenMP, the Structural Simulation Toolkit (SST), MESI/MOESI cache coherence protocols, HPC batch job automation, genetic algorithms (DEAP)

**Tools and Platforms:** Git and GitHub, Google Colab, RStudio and Posit Cloud, Xcode, OpenRocket
